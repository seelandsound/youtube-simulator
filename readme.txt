==================================================
==        YouTube 企劃模擬器 - 說明文件         ==
==================================================

[專案簡介]

這是一個雲端協作工具，旨在讓內容創作者模擬 YouTube 介面，
測試不同影片企劃的受歡迎程度。


[核心架構]

本專案採用「前端渲染」搭配「Google Apps Script 作為無伺服器後台」的架構。

  - 前端 (index.html, stats.html): 
    使用者介面，負責渲染與互動。

  - 後台 (Google Apps Script): 
    作為唯一的「單一事實來源」，綁定至 Google Sheet。


--------------------------------------------------

[後台部署指南 (Google Apps Script)]

這是讓整個專案能成功運作的關鍵。請將以下程式碼部署為一個網路應用程式。

[步驟]

  1. 在 Google Sheet 中，點擊「擴充功能」 > 「Apps Script」。
  2. 將下方完整的程式碼貼入編輯器。
  3. 點擊「部署」 > 「新增部署」。
  4. 設定：
     - 類型: 網路應用程式
     - 執行身份: 我
     - 誰可以存取: 任何人
  5. 點擊「部署」，並複製產生的網址。
  6. 將此網址更新至 index.html 和 stats.html 的 SCRIPT_URL 常數中。


[Apps Script 程式碼]

  // =======================================================
  // ==  請將以下所有程式碼複製到您的 Apps Script 編輯器中  ==
  // =======================================================

  const SHEET_ID = '【請貼上您的 Google Sheet ID】';
  const DB_SHEET_NAME = 'VideoDatabase';

  // [ 此處請貼上您完整的、最終可以運作的 Apps Script 程式碼 ]
  // (doGet, doPost, 以及所有輔助函數...)
  

  // =======================================================
  // ==                      程式碼結束                     ==
  // =======================================================


[重要提醒]

  請務必將上方的 '【請貼上您的 Google Sheet ID】' 替換成您自己的 Sheet ID，
  並將完整的 Apps Script 程式碼填入。

  這樣，未來的您或任何協作者，都能快速地重新部署這個專案。
